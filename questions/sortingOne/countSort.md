# Q2. Count Sort

**Problem Description**  
Given an array **A**, sort this array using the Count Sort Algorithm and return the sorted array.

**Problem Constraints**
- `1 <= |A| <= 10^5`
- `1 <= A[i] <= 10^5`

**Input Format**
- A single integer array **A**.

**Output Format**
- Return the sorted array of integers.

### Explanation 1:

---

### Input 2:
```plaintext

```
### Output 2:
```plaintext
[1, 2, 3, 4]
```

Array sorted using Count Sort is `[1, 2, 3, 4]`.
